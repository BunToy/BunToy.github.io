> css 设计，当做笔记记之。

### DIV置于最底层

- css样式

```ruby
 style="z-index:-100;position: relative;"
```

仅仅在position: relative; 或者 绝对布局使用z-index 才会生效

### DIV置于最高层

- CSS样式
```ruby
 style="z-index:9999;position: relative;"
```
